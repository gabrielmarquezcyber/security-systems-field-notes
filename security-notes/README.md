# Security Notes

This folder is for public, defender-focused security notes.

Examples:

- SOC investigation paths
- SIEM query notes
- detection logic explanations
- alert triage frameworks
- vulnerability prioritization notes
- identity and cloud security notes
- sanitized evidence handling examples

Avoid:

- exploit walkthroughs
- offensive command dumps
- private lab answers
- screenshots with secrets, flags, tokens, user data, or private infrastructure
