# Public Proof Artifact Checklist

Use this before publishing any public artifact.

## Strategic Fit

```text
[ ] Does this feed Income Activation, Cognitive Arsenal, or AI Trust Architecture?
[ ] Does this prove something a recruiter, hiring manager, or technical reviewer can inspect?
[ ] Is the artifact role-relevant rather than merely interesting?
```

## Evidence Quality

```text
[ ] Scenario is clear
[ ] Evidence source is clear
[ ] Decision logic is clear
[ ] Limitations are stated
[ ] False positives or uncertainty are acknowledged
[ ] No private data, secrets, tokens, internal-only material, or sensitive screenshots
```

## Tone

```text
[ ] Grounded
[ ] Not grandiose
[ ] Not bitter
[ ] Not contemptuous toward recruiters, employers, women, institutions, or candidates
[ ] Does not pretend lab work is production experience
```

## Distribution

```text
[ ] GitHub version exists
[ ] LinkedIn excerpt drafted
[ ] One-sentence recruiter/interviewer utility statement exists
[ ] Repo/profile can link to it cleanly
```
